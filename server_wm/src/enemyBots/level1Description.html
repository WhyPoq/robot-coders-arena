Level 1

Hello, this is Robot Coders Arena.

Write code in the black area on the left.

Below this guide you can see your enemy's code as an example.

Every turn the code inside move function is run. Whatever this function returns will be the action the bot executes.

For example, in this level your enemy just upgrades every move

For now just remember these simple rules:
  • ATTACK beats UPGRADE
  • BLOCK beats ATTACK
  • UPGRADE beats BLOCK

Later you will learn more.

For now just write your bot's code and press the red play button on the top to start the fight
