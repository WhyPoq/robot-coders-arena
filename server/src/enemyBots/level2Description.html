Level 2
Each time the move function is run it gets some parameters.

There are plenty. To beat this level it is enough to now only 1 of them:

shortMemory - array of 10 numbers filled with 0's by default. If you update this array inside move function, your changes will persist for the next calls of move function.
It is like memory of your robot for the round.

And more about actions:
  • If you BLOCK the ATTACK of the enemy, they get stunned for 2 moves
  • If you UPGRADE, you increase your strength (attack) by 1. However, if during upgrading you get hit, you get double damage from the enemy

You can learn the complete details about actions by clicking on the Info tab
